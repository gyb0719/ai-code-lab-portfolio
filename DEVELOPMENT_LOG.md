## 🎯 프로젝트 개요
- **프로젝트명**: AI 코드 연구소 포트폴리오
- **개발자**: 권용범
- **개발 기간**: 2024년 12월
- **목적**: 프리랜서 개발자로서의 역량을 효과적으로 보여주는 포트폴리오 웹사이트 구축

## 🛠 기술 스택 선정 이유

### Next.js 14 (App Router)
- 최신 React 기능 활용 가능
- 서버 컴포넌트로 초기 로딩 속도 개선
- Vercel 배포 최적화
- SEO 친화적인 구조

### TypeScript
- 타입 안정성으로 런타임 에러 방지
- 개발 생산성 향상
- 코드 자동완성 및 IntelliSense 지원

### Tailwind CSS
- 빠른 스타일링
- 일관된 디자인 시스템
- 커스텀 네온 테마 구현 용이

### Framer Motion & Three.js
- 부드러운 애니메이션 효과
- 3D 인터랙티브 요소로 차별화
- 사용자 경험 향상

### 애니메이션 전략
- 과도하지 않은 마이크로 인터랙션
- 60fps 유지를 위한 transform/opacity 활용
- will-change 속성 최소화

## 💡 학습된 내용

### 기술적 인사이트
1. **Three.js 최적화**: 불필요한 리렌더링 방지를 위한 메모이제이션
2. **Framer Motion**: AnimatePresence로 언마운트 애니메이션 구현
3. **Next.js 14**: Server Components와 Client Components 적절한 분리
4. **TypeScript**: 제네릭과 유틸리티 타입 활용

### 문제 해결 경험
1. **문제**: 3D 요소로 인한 성능 저하
   - **해결**: requestAnimationFrame 최적화, 파티클 수 조절

2. **문제**: 모바일에서 커스텀 커서 오작동
   - **해결**: 터치 디바이스 감지 후 비활성화

3. **문제**: Tailwind 커스텀 애니메이션 적용 안됨
   - **해결**: tailwind.config.ts에 keyframes 정의
